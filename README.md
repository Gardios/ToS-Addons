# Addons
## Advanced Skill Tooltips
This addon expands skill tooltips with more information. The added information is:
* Skill level
* SP cost (base and adjusted)
* Cooldown (base and adjusted)
* Target
* Execution time
* Cancel time
* Required weapons
* Jumping usability
* Riding usability
* Available attributes (active/inactive)
